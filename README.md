## Fitur Utama

- **Responsive Design**: Proyek ini sepenuhnya responsif dan dapat diakses di berbagai perangkat.
- **Dark Mode**: Tersedia opsi untuk beralih antara mode terang dan gelap.
- **Animasi Halus**: Menggunakan Framer Motion untuk memberikan pengalaman pengguna yang lebih interaktif.
- **Komponen Modular**: Setiap bagian dari portofolio dibangun sebagai komponen terpisah untuk kemudahan pemeliharaan.

## Cara Menjalankan Proyek

1. Clone repositori ini ke mesin lokal Anda.
2. Jalankan `npm install` untuk menginstal semua dependensi.
3. Jalankan `npm run dev` untuk memulai server pengembangan.
4. Buka browser dan akses `http://localhost:3000` untuk melihat portofolio Anda.
